# Приёмка проекта

## Код

- [x] Next.js 14 App Router.
- [x] TypeScript strict.
- [x] Ant Design 5.
- [x] Нет `@supabase/ssr`.
- [x] Middleware не участвует в авторизации.
- [x] Кастомные SVG-иконки 24×24.
- [x] Production build проходит.

## Авторизация

- [x] Табельный номер преобразуется в технический email.
- [x] Телефон нормализуется в международный формат.
- [x] Используется прямой password grant Supabase Auth REST API.
- [x] Токены сохраняются в localStorage.
- [x] Refresh token обновляет сессию до истечения.
- [x] Серверные компоненты читают проверенную cookie через `next/headers`.
