# Реализация фирменного стиля

Интерфейс следует правилам брендбука «Бионит» 2026:

- основная палитра: белый `#FFFFFF`, тёмный `#1D1D1B`, акцентный красный `#CB342A`;
- дополнительный бордовый `#A71916`, розовый `#F48C8C`, серые оттенки;
- фирменный шрифт Geologica;
- визуальное правило 60:30:10;
- геометрический капсульный паттерн;
- красные круги, овальные карточки и мягкие радиусы как отсылка к знаку и лекарственной капсуле;
- контрастные монохромные SVG-иконки;
- отсутствие эмодзи в интерфейсе;
- аккуратное охранное поле вокруг логотипа.

Файлы фирменной графики находятся в `public/brand/`. Все прикладные иконки реализованы как React SVG в `src/components/Icon.tsx`.
